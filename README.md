# Trial Manager

## Setting Up

* Configure Global Variables
 * The path to the right of 'php_value auto_prepend_file' is for the config.php file. Make sure it's valid for your server.
 * In js/global_var.js, make sure gVar.root is set to the root directory of this site RELATIVE to the root directory of your web server. By default, this variable is set to the trial_mgr directory in the server's root directory.
